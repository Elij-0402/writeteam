# WriteTeam - Project Overview

**Date:** 2026-03-01  
**Type:** web  
**Architecture:** monolith / layered

## Executive Summary

WriteTeam 是一个面向小说创作场景的全栈 AI 写作应用，支持项目与文档管理、富文本编辑、Story Bible、Canvas 与多类 AI 写作辅助能力。

## Project Classification

- **Repository Type:** monolith
- **Project Type:** web
- **Primary Language:** TypeScript
- **Architecture Pattern:** Next.js App Router + Supabase

## Technology Stack Summary

- Next.js 16, React 19, TypeScript 5
- Tailwind CSS v4 + shadcn/ui
- Supabase + SQL migrations
- Vitest + Node contract tests

## Key Features

- AI 写作：Write/Rewrite/Expand/Describe/Brainstorm/First Draft
- AI Chat 与 Muse
- Story Bible 与系列管理
- Canvas 创意结构编辑
